# Weather App

## Objective
This project fetches and displays real-time weather information for a user-specified city using the OpenWeatherMap API.

## Features
- Search weather by city name
- Display temperature
- Display humidity
- Display weather condition
- Display wind speed
- Handle invalid city names

## Technologies Used
- Python
- Requests Library
- OpenWeatherMap API

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python Weather-app.py
